# Frontend - CarSell (HTML, CSS e JavaScript)

Este frontend foi refatorado para ser um site estático em **HTML, CSS e JavaScript puro**.
Não há mais React, Expo, Vite ou dependências de build.

## Estrutura do Projeto

```
frontend/
├── index.html
├── style.css
├── app.js
└── README.md
```

## Como usar

1. Abra `frontend/index.html` diretamente no navegador.
2. Ou use um servidor local simples, como `Live Server` no VS Code.

## Funcionalidades

- Catálogo de carros exibido em cartões responsivos
- Busca por marca, modelo, ano e faixa de preço
- Tela de login e cadastro local usando storage do navegador
- Tela para anunciar seu carro e publicar no catálogo
- Layout mobile-first
- Sem React, sem Expo, sem dependências externas

## Observações

- Se quiser usar um servidor local, abra a pasta `frontend` no VS Code e use a extensão `Live Server`.
- O código está preparado para ser estendido com novas páginas e funcionalidades em JavaScript puro.
